## 🧭 About Me

I'm a **Techno-Functional Consultant** working at the intersection of core banking systems and software engineering. My day job is building, customizing, and integrating modules on **Oracle FLEXCUBE** and **OBB** for banks running both **conventional and Islamic** operations. My side mission is automating the parts of the job that shouldn't have to be done by hand.

This page is a tour of *what I do* and *how I work* — not a list of every project I've ever shipped. For that, my CV does the heavy lifting.

### 🏦 Core Banking Modules
End-to-end module delivery on FLEXCUBE and OBB — from BRD analysis through design, development, UAT, and go-live. Comfortable across **PL/SQL**, **Java**, **JavaScript**, and **OJET**. ### 🔌 Web Services & Integrations

When a workflow takes longer than it should, I reach for **Python**, **Playwright**, or **AI tooling** to bring it down. Built a Playwright suite for post-patchset regression, a Claude Code skill that drops RAD generation from 2 days to 10 minutes, and a SAM screen that replaced manual generation of 107 statutory reports.

</td>

Deployment verification, invalid-check resolution, go-live configuration, and post-live support. Including the team's **first international bank deployment** — handled cutover-day work end to end.
